# Memory Maintenance SOP

更新时间：2026-03-01

## 日更（每日至少 1 次）
1. 将当天关键信息写入 `memory/YYYY-MM-DD.md`
2. 只记录：决策、偏好变化、失败经验、待办
3. 临时碎片优先写 Memos（便于后续检索）

## 周更（每周 1 次）
1. 回看最近 7 天 daily memory
2. 抽取长期有效内容，写入 `MEMORY.md`
3. 清理过期临时信息（避免冗余）

## 记忆分层
- 热记忆：daily + MEMORY（高相关）
- 温记忆：Obsidian 知识库
- 冷记忆：Memos 历史碎片

## 检索路由
1. 热记忆（本地）
2. 再查 Memos
3. 仍不足再外部检索
