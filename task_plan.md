# 河南六大重点领域项目机会监控任务计划

## 目标
按固定搜索路由，对河南省厅办、重点地市、驻豫国央企在六大领域的新增动作进行发现与确认，输出可转化项目机会简报。

## 阶段
- [complete] 阶段1：检索上一轮上下文与今日候选
- [complete] 阶段2：正文确认与补抓
- [complete] 阶段3：筛选高价值新增并形成机会判断
- [in_progress] 阶段4：输出定时报告

## 约束
- 发现层与确认层分离
- 优先A类源，其次B类源，C类仅补线索
- 宁缺毋滥，每条必须带链接与状态
