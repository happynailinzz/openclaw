# 南阳 AI 行动方案网页包（可上站版本）

## 建议站点路径

- 下载页：`/nanyang-ai-plan-2026/`
- 在线预览页：`/nanyang-ai-plan-2026/view/`
- PDF 直链：`/downloads/nanyang-ai-plan-original.pdf`

## 目录说明

- `nanyang-ai-plan-2026/index.html`：下载页
- `nanyang-ai-plan-2026/view/index.html`：在线预览页
- `downloads/nanyang-ai-plan-original.pdf`：PDF 文件

## 部署方式

将本目录中的：

- `nanyang-ai-plan-2026/`
- `downloads/`

同步到站点根目录即可。

## 备注

当前页面为静态 HTML，不依赖构建工具，适合直接上传到 Nginx / 静态站点目录。
