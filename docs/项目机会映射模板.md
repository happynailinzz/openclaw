# 项目机会映射模板

## 用途
把“新闻 / 政策 / 公告 / 项目动态 / 产业动作”快速转成可判断、可跟踪、可沟通的项目机会结构。

适用场景：
- 河南项目机会简报
- 政企数字化 / AI / 能源数字化选题
- 对外咨询判断
- 客户沟通前的快速分析

---

## 一、标准映射结构

### 1. 政策 / 动作
回答：到底发生了什么？

填写要点：
- 谁发的？
- 发了什么？
- 是文件、试点、平台、招标、签约、投资、投产，还是调研表态？
- 属于新增，还是旧事重提？

示例：
- 河南省通信管理局将“算力、人工智能发展、5G规模化应用”列入 2026 年重点工作。

---

### 2. 对应场景
回答：这件事会落在哪类实际场景？

常见场景：
- 政务数字化
- 园区平台
- 行业数据治理
- 智算中心运营
- 制造业 AI
- 智慧教育 / 实训
- 医疗 AI
- 物流调度
- 源网荷储 / 虚拟电厂 / 能碳平台

示例：
- 园区算力调度、行业 AI 应用示范、5G+工业互联网改造。

---

### 3. 项目形态
回答：它最可能长成什么项目？

常见项目形态：
- 平台建设
- 系统集成
- 数据治理工程
- 行业应用场景包
- 咨询规划 + 顶层设计
- 实训基地 / 课程平台
- 算力运营 / 服务采购
- 设备采购 / 招标实施
- 运维与持续服务

示例：
- 城市 / 园区算力调度平台
- AI 场景集成项目
- 数据标注与治理平台

---

### 4. 可能牵头单位
回答：谁最可能是项目主导方？

类型可包括：
- 省厅局
- 地市政府
- 园区管委会
- 平台公司
- 国央企区域公司
- 高校 / 职校 / 医院 / 国企用户单位

示例：
- 郑州航空港管委会
- 空港数字城开
- 省发改 / 工信 / 政数系统

---

### 5. 合作对象 / 供应侧角色
回答：谁可能参与做这件事？

类型可包括：
- 咨询公司
- 系统集成商
- 算力厂商
- 数据治理厂商
- 模型厂商
- 运营商
- 高校与培训机构
- 场景应用供应商

示例：
- 科大讯飞、超聚变、运营商、系统集成商、数据治理服务商

---

### 6. 资金口径
回答：钱可能从哪来？

常见口径：
- 财政专项
- 专项债
- 园区建设资金
- 平台公司投资
- 企业 CAPEX
- 场景分期实施预算
- 服务费 / 运营费
- 就业培训专项
- 示范申报资金

示例：
- 平台建设费 + 场景应用包 + 后续运营服务费

---

### 7. 当前判断等级
建议分四档：
- A：高价值硬机会（已见明确项目抓手）
- B：高概率机会（方向明确，等进一步落地）
- C：观察机会（有信号但证据不够）
- D：噪音 / 不升格

---

## 二、推荐输出格式

### 模板 A：简版（适合简报）
**政策 / 动作：**

**对应场景：**

**项目形态：**

**可能牵头单位 / 合作对象 / 资金口径：**

**判断等级：**

**来源链接：**

---

### 模板 B：扩展版（适合方案前分析）
**1. 发生了什么**

**2. 为什么这不是空消息**

**3. 最可能落在哪类场景**

**4. 最可能长成什么项目**

**5. 谁会牵头，谁会参与**

**6. 钱可能从哪里出**

**7. 现在该继续盯什么**

---

## 三、过滤规则
以下内容默认不直接升格为项目机会：
- 纯口号式表态
- 没有主体、没有抓手、没有场景的泛政策
- 没有新增信息的旧闻重复
- 单纯媒体转述、没有官方或权威确认
- 只有概念、没有预算 / 平台 / 场景 / 试点 / 招标 / 投资入口

---

## 四、优先级判断口诀
优先判断四件事：
1. **谁发的**
2. **谁给钱**
3. **谁牵头**
4. **谁落地**

如果这四件事里能回答两件以上，基本就值得继续追。

---

## 五、适合当前业务的重点映射方向

### 1. 政企数字化
重点看：
- 政务平台
- 数据治理
- 一网统管 / 一网通办
- 国企数字化改造

### 2. 算力 / 数据中心 / 信创
重点看：
- 智算中心
- 数据中心配套
- 算力调度
- 国产化替代
- 平台型建设项目

### 3. AI 应用落地
重点看：
- 行业赋能中心
- 垂直模型
- 场景示范
- 训练 / 推理 / 集成平台

### 4. 源网荷储 / 绿色储能 / 风光
重点看：
- 虚拟电厂
- 绿电交易
- 负荷聚合
- 园区能碳平台
- 配储与协同调度

### 5. 智慧教育 / 产教融合
重点看：
- 实训平台
- 课程平台
- 校企合作
- 能力评估
- 人才供给体系

---

## 六、一句话版本
后续看到任何高价值线索，都不要停在“这新闻挺重要”，而要继续追问：
**它会变成什么项目，谁来做，谁出钱。**
